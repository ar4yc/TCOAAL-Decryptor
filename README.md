# TCOAAL Decryptor

#### ⚠ Note: This project is currently a work in progress.
###### Requires to own the game to work.

A tool to decrypt [_The Coffin of Andy and Leyley_](https://store.steampowered.com/app/2378900/The_Coffin_of_Andy_and_Leyley/) (by _Nemlei_) and restore the original filenames.
